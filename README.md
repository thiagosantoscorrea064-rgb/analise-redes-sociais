# Análise de Redes Sociais

Projeto desenvolvido para a disciplina de Laboratório de Programação (2026/1).

## Objetivo

Implementar uma análise de redes sociais utilizando três paradigmas de programação:

- Python (Orientação a Objetos)
- Common Lisp (Programação Funcional)
- Prolog (Programação Lógica)

## Estrutura do projeto


## Funcionalidades

- Cadastro de usuários
- Relações de seguidores
- Amigos diretos
- Amigos em comum
- Grau de separação
- Sugestão de conexões
- Ranking de usuários influentes

Cada implementação utiliza o mesmo conjunto de dados para permitir comparação entre os paradigmas.
